# UCI-CS241-Advanced-Compiler-main 2023 Winter
1. This project is a humble implementation of DLX SSA-based compiler for the course CS241 Advanced Compiler Class taught by Professor Michael Franz in 2023 winter at UC Irvine. The goal of realsing this project is to help future students with having better learning experience about compiler construction.
2. For running the compiler project, python version needs to be > python3.7+.
3. Since this project is written individually, coloring and functions for the compiler are not implemented. This compiler includes basic algebra operation compilation, if statement, while statement, and array data structure handlement.
4. For running the code and test the results, please do > python main.py.
5. For testing the compiler with different test code written in dlx, you can add new files in txt file and save it to the test directory. Then change the file name in main.py.
